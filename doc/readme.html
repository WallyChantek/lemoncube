FPS Limiter

Overview

Abstract:

The FPS Limiter class provides a means by which to cap the game's maximum framerate.

Information:

The FPS Limiter is not mandatory to develop games with LemonCube. However, if you're like me and do not favor variable timesteps, you may find it preferable to use this class.

Basic Usage:

  1. Create a new FpsLimiter object, providing your desired framerate (I cap it at 60 FPS for my games).
  2. Call the prepare() function at the beginning of love.update().
  3. Call the limitFps() function at the end of love.draw().
